#### 新添加的指令
模仿原文凯尔希的字体（使用网上的行书字体模仿，不完全相同甚至差别挺大）：
- `\KalWrite`: 作用于一个段落/域
- `\Kal{}`: 调用小段的文字

侧边注释：
- `mnote{}`: 注释，对齐指代的文字

- Epigraph 环境: 生成原书每张开头的段落，目前没有完全按照原书（太复杂了不会做）
